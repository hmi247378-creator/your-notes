# 你的笔记（Your Notes）

本仓库为“你的笔记”Phase 1 MVP 实现（PC Web + 微信小程序 + 统一后端）。

## 快速开始（本地开发）

### 1) 安装依赖
在仓库根目录执行：
```bash
npm install
```

### 2) 数据库（默认 SQLite，可选 Postgres/Redis）

**默认**：后端用 SQLite（无需 Docker），开箱即跑。

**可选**：如果你安装了 Docker，可启动 Postgres/Redis：
```bash
npm run db:up
```

### 3) 配置环境变量
复制并填写：
- `services/api/.env`（参考 `services/api/.env.example`）
- `apps/web/.env.local`（参考 `apps/web/.env.example`）

### 4) 初始化数据库
```bash
npm run prisma:generate
npm run prisma:migrate
```

### 5) 启动 API + Web
```bash
npm run dev
```

默认地址：
- Web：`http://localhost:3000`
- API：`http://localhost:3001/api`
